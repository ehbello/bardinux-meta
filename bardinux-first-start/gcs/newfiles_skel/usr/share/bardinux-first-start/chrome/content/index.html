Bienvenido a Bardinux

Con Bardinux, tendrás un sistema operativo completo y funcional en cuestión de minutos.

Bardinux incluye gran cantidad de software por defecto, como una suite ofimática, reproductores de música y vídeo, navegador web, lector de correo y mucho más.

Pero además Bardinux, está pensada para ser usada por la comunidad universitaria, por lo que provee conjuntos de programas usados en las distintas carreras.
