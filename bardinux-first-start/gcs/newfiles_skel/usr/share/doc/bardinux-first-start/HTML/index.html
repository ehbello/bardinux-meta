¿Qué es Bardinux?

El software que necesitas al alcance de tu mano

Bardinux es una adaptación de la distribución de Linux 'kubuntu' a las necesidades en cuanto a Software de los miembros de la comunidad universitaria de la ULL. Se ha recopilado el software más relevante para la realización de prácticas, investigación y uso general, en paquetes fácilmente instalables.

¿Como empiezo a utilizar Bardinux?

Bardinux puede utilizarse tanto sin instalarse (introduciendo el CD y encendiendo el ordenador), o bien instalándolo.

Mirando las distintas secciones de esta página web puedes encontrar información básica de como comenzar con Bardinux.

Si necesitas más información, en la sección de soporte encontrarás diversos enlaces donde encontrarás ayuda.

Acerca de la oficina del Software Libre

La Oficina de Software Libre de la ULL es el ente de la Universidad creado para promover el uso de Software Libre y estándares abiertos. Además de Bardinux, se han creado múltiples proyectos, como el correo del alumnado, el disco duro virtual, y otros destinados a la formación, como el CILA, cursos para el profesorado y el PAS.
